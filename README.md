# Deep Dive - Os primeiros cristãos

## 📒 Descrição
Com base em um trecho de um livro sobre História da Igreja, mais precisamente sobre o início dela, criar um podcast entre duas ou três pessoas falando livremente sobre aquele trecho.

## 🤖 Tecnologias Utilizadas
Gemini Advanced 
https://clideo.com/pt/video-translator 
Youtube
Wondershare Filmora

## 🧐 Processo de Criação
Gemini Advanced para criar o áudio simulando um podcast entre duas ou três pessoas. O script que eu usei foi "Criar Visão Geral em Áudio", sugerida pela própria IA, anexei o arquivo com o trecho do livro e consegui o áudio.
Com o https://clideo.com/pt/video-translator eu criei uma legenda em Português-BR para poder colocar no vídeo.
Do Youtube peguei o trecho de um vídeo de Israel histórica.
Com o Wondershare Filmora eu uni todos esses elementos para criar o video final.

## 🚀 Resultados
Um áudio muito fluído, bastante natural, muito próximo do real, não fosse os "vícios de linguagem" fora de contexto que a IA inseriu no áudio para dar um tom de naturalidade, estaria perfeito, na minha visão.

## 💭 Reflexão (Opcional)
Minha primeira ideia era criar um pequeno livro para crianças, com imagens consistentes e próximo da criação real de uma pintura no estilo aquarela.
